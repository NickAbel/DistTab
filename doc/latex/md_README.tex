Dist\+Tab is a library being developed for use with combustion codes that use spray flame lookup tables. Currently, simulations across multiple NUMA nodes store the entire flame lookup table on each node, which limits the size of the lookup table. Dist\+Tab distributes the lookup table across nodes, partitions the table in a logical manner and uses MPI one-\/sided communication to retrieve lookup table values and stores the partitions containing the lookup values on the local node using a FIFO queue.\hypertarget{md_README_autotoc_md3}{}\doxysubsection{Functionality}\label{md_README_autotoc_md3}

\begin{DoxyItemize}
\item Lookup table storage
\begin{DoxyItemize}
\item Loading/reading in table
\item Reorganizing table into partitions based on locality in thermochemical state space
\end{DoxyItemize}
\item MPI get values from other nodes
\begin{DoxyItemize}
\item Based on partition, stored in FIFO queue locally
\end{DoxyItemize}
\item And more...
\end{DoxyItemize}\hypertarget{md_README_autotoc_md4}{}\doxysubsection{Prerequisites}\label{md_README_autotoc_md4}
